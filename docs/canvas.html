HTML 5 Canvas

HTML 5 introduced support for a canvas element that can be programmatically drawn on. The really nice thing is that it is very similar to drawing using Processing (in fact, p5.js relies heavily upon the canvas element).

Basic Usage Example

<canvas width="600" height="600" id="mycanvas" />

<script type="text/javascript">
	var canvas = document.getElementById('mycanvas');
	var context = canvas.getContext('2d');
	context.fillStyle="#FF0000";
	context.fillRect(0,0,canvas.width,canvas.height);
</script>
